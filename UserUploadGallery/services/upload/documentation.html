1. jQuery File Upload Demo
  2. Download library here
  3. Copy all .js files in js directory from the downloaded directory to your directory
  4. Copy .css files in css directory from the downloaded directory to your directory:
    jquery.fileupload.css, jquery.fileupload-noscript.css, jquery.fileupload-ui.css, jquery.fileupload-ui-noscript.css
  5. Copy images from the img directory and make sure that they will have the same absolute path
  6. Copy a form with id="fileupload" in the file index.html to the place where you want to upload files
  7. Include all the above .css and .js files.
    Also include src="//blueimp.github.io/JavaScript-Load-Image/js/load-image.all.min.js" (Load Image plugin is included for the preview images and image resizing functionality)
  8. Copy to you directory php scripts from \server\php. Be careful with correct paths in your code!
  9. In main.js you can change all your own settings
  10. Copy from index.html scripts with id="template-download", id="template-upload" .
    They are responsible for rendering files list in the widget
  11. Now you are able to test uploading
  12. In case some problems, read documentation attentively
  1. assets/js/upload/main.js == Initialize the jQuery File Upload widget
  2. assets/js/upload/jquery.fileupload.js == upload core library
  3. assets/js/upload/tmpl.min.js == Load Image plugin is included for the preview images and image resizing functionality
  4. assets/js/upload/jquery.fileupload-validate.js == image type/size/.. validation for uploading
  5. assets/js/upload/jquery.fileupload-process.js == fileupload widget (user interface core)
  6. assets/js/upload/jquery.fileupload-process.js == the File Upload Processing plugin extends the fileupload widget with file processing functionality (for progressbar)
  7. assets/js/upload/jquery.fileupload-image.js == one of additional plugin of the jQuery File Upload widget, in particular for uploading images
  8. assets/js/upload/vendor/jquery.ui.widget.js == The Templates plugin is included to render the upload/download listings
  9. assets/css/upload/jquery.fileupload.css == main css for jQuery File Upload widget
  10. assets/css/upload/jquery.fileupload-noscript.css == CSS adjustments for browsers with JavaScript disabled
  11. services/upload/ImageUploader.php == server-side script initialization for jQuery File Upload
  12. services/upload/UploadHandler.php == the class with all logic for uploading fles to the server
